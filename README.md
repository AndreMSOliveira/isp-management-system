# ISP Management System

Sistema Full Stack para gerenciamento de clientes.

## Tecnologias
- Node.js
- Express
- MySQL
- HTML
- CSS
- JavaScript

## Funcionalidades
- Cadastro de clientes
- Listagem de clientes
- Edição de clientes
- Exclusão de clientes
- Busca por nome
- Dashboard com métricas

## Como executar
npm install
npm start
